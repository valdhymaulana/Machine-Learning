Kesimpulan pengerjaan modul Week 3:
- Outlier Handling: Analisis distribusi dan pemotongan outlier ekstrim dengan IQR.
- Missing Value: Drop kolom rusak >20%, serta imputasi data menggunakan Median (numerik) dan Modus (kategorik).
- Categorical Encoding: Cleaning teks redundan, lalu konversi data menggunakan One-Hot, Label, dan Mean Encoding.
- Pencegahan Leakage: Semua manipulasi statistik dieksekusi setelah train_test_split (80:20).
